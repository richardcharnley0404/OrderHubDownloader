# Icons

This directory contains icon files for the OrderHub Downloader application.

## Required Files

- **icon.ico** - Windows tray icon (16x16 recommended for tray, but should include multiple sizes)
- **icon.png** - Application icon (256x256 or larger)

## Creating Icons

You can create icons using:
- Online tools like: https://www.favicon-generator.org/
- Desktop tools like GIMP or Photoshop
- Icon creation software

For now, you can use any .ico file. The application will use the system default icon if these files are not present.

## Quick Setup

To get started quickly:
1. Find any .ico file (e.g., from C:\Windows\System32\*.ico)
2. Copy it to this directory as `icon.ico`
3. Or create a simple icon online and place it here
